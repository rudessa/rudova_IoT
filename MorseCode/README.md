# Техническое задание
## Система передачи азбуки Морзе через свет

---

## Что делает система

Два Arduino передают друг другу сообщения азбукой Морзе. Первый использует свет для передачи, второй расшифровывает и показывает символы на дисплее.

---

## Первый Arduino - Передатчик

**Что нужно:**
- Arduino
- Сервопривод
- Фоторезистор

**Как работает:**
- Сервопривод закрывает и открывает фоторезистор
- Фоторезистор видит изменения света (это точки и тире Морзе)
- Arduino читает эти сигналы
- Отправляет данные второму Arduino через UART

---

## Второй Arduino - Приёмник

**Что нужно:**
- Arduino
- Семисегментный дисплей

**Как работает:**
- Получает данные от первого Arduino через UART
- Переводит азбуку Морзе в обычные символы
- Показывает символы на дисплее по очереди

---

## Требования

**Фильтрация**
- Убрать шумы и помехи при чтении с фоторезистора

**Что передаём**
- Цифры: 0-9
- Буквы: A-Z (английские)

**Связь**
- Два Arduino соединены через UART

**Дисплей**
- Символы показываются по одному на семисегментном индикаторе
