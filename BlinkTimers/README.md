# BlinkTimers

---

| `timer2_blink.ino` | Задание 1 | Blink по прерываниям Timer2 (~2 сек) |
| `millis.ino` | Задание 2 | Своя реализация функции millis() |
| `three_models.ino` | Задание 3 | Три режима blink с управлением по UART |
| `analog_poll.ino` | Задание 4 | Опрос аналогового пина каждые 250 мс |

---

## Подключение

```
Arduino Pin 2 → Резистор 220Ω → LED (анод +)
LED (катод -) → GND

```

**Важно:** Соблюдайте полярность светодиода!

---

## Подключение

1. Подключите светодиод к пину 2 через резистор 220 Ом
2. Откройте нужный файл `.ino` в Arduino IDE
3. Загрузите код на плату (Upload)
4. Откройте Serial Monitor (Ctrl+Shift+M, 9600 baud)
5. Для Задания 3: отправляйте команды `1` или `2`

---

## Отладка

| Проблема                        | Решение                                     |
| ------------------------------- | ------------------------------------------- |
| LED не мигает                   | Проверьте полярность LED и резистор 220 Ом  |
| Serial не показывает данные     | Проверьте скорость 9600 baud                |
| Команды в Задании 3 не работают | Отправляйте только `1` или `2` без пробелов |
